核心功能

盤中即時監控

交易時段（09:00–13:30）每 60 秒抓取一次委賣一價格，與監控清單比對後觸發訊號。

雙行情來源

TWSE MIS（主）+ Yahoo Finance HTML（副），Freshness-First 規則以 tick_at 較新者為準。

5 分鐘冷卻機制

同 stock_no + stock_status 組合 5 分鐘內不重複推播，避免訊號疲勞。

LINE 模板化通知

所有 LINE 訊息（彙總/摘要/測試）均由 template_key + context 渲染，不硬編碼文案。

每日 14:00 估值

三種估值模型（艾蜜莉/老牛/雷司紀）逐股運算，結果寫入 SQLite，隔日監控即可使用。

補償機制

LINE 成功 + DB 失敗時，寫入 pending_delivery_ledger，補償完成前視同已通知，避免重複推送。

快速啟動

環境設定
# 1. 安裝相依套件
pip install -r requirements-dev.txt

# 2. 設定 LINE 環境變數
$env:LINE_CHANNEL_ACCESS_TOKEN = "your_token"
$env:LINE_TO_GROUP_ID          = "your_group_id"

# 3. 啟動守護行程
python -m stock_monitor run-daemon

# 4. 執行全部測試
python -m pytest -q tests
Windows 排程（開盤/收盤自動啟停）
# 一次性註冊 Windows 工作排程器
# 08:50 自動啟動 daemon（週一至週五）
# 14:30 自動停止 daemon（週一至週五）
powershell -ExecutionPolicy Bypass -File scripts/register_scheduled_tasks.ps1
